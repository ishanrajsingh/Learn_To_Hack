Part 1: Directory & File Manipulation ->
#1 mkdir newDirectory
	This command create a new directory in current folder named as newDirectory
#2 cd newDirectory
	This command let us change our current working directory to newDirectory.
#3 touch harshit.txt
	This command create a text file named harshit.
#4 mv harshit.txt IIT2025003.txt
	This command change name of harshit.txt to IIT2025003.txt
#5 cd ..
	This command let us step out of current working directory to its parent folder
#6 rm -rf newDirectory
	This command delete all content of newDirectory including itself.
